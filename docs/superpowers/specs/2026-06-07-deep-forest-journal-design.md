# 股票日记深海森林视觉设计

## 目标

把现有股票日记从工具型深色仪表盘，重构为具有私人投资手账感的移动优先界面。保留所有现有功能、数据结构、路由和接口。

## 视觉语言

- 基底：深海军蓝与黑翡翠，营造安静、私密、长期记录的氛围。
- 主强调：荧光青柠，仅用于主操作、选中状态和关键数字。
- 辅助色：薄荷绿、森林绿、青蓝、淡紫，用于区分卡片与状态。
- 字体：中文衬线字体承担标题和判断文本，无衬线字体承担操作与说明，等宽字体承担 ticker、价格、分数和标签。
- 形状：18-28px 大圆角卡片，减少硬边框，以色块、间距和轻阴影建立层级。
- 质感：使用 CSS 径向光晕与细网格，不使用参考图片作为背景，不引入版权不明的素材。

## 页面设计

### 首页

- 顶栏成为深色胶囊式工具条。
- Hero 使用更短、更有编辑感的标题和柔和绿色光晕。
- 统计条改为四张独立小卡。
- 股票日记本改为错落色块卡片，ticker 与当前心境成为首要信息。
- 新建入口使用荧光青柠色，成为唯一高强度 CTA。

### 新建流程

- 步骤条改为圆角胶囊。
- 操作类型、搜索结果、问题选项改为可触摸的大圆角块面。
- 问题页保持单题聚焦，选中态使用青柠高亮。
- 结果页使用大号状态字和柔和彩色摘要卡。

### 股票详情

- 股票信息区改为主档案卡。
- 行情、估值和心境轨迹使用独立色块分组。
- Thesis 卡使用深森林绿，便条按心境使用薄荷、黄、橙、红色阶。
- 操作按钮保持可辨识，但降低装饰性边框噪声。

### 移动端

- 以 390px 宽度为主要设计基准。
- 保持按钮最小触摸高度 44px。
- 卡片单列，统计区两列，搜索栏独占一行。
- 弹窗从居中窗口转为接近底部抽屉的布局。

## 不变范围

- `journal_v1` localStorage 数据格式不变。
- Hash 路由不变。
- `/api/search`、`/api/quote`、`/api/profile` 不变。
- 心境题目、评分、导入导出和删除逻辑不变。
- 本轮不拆分单文件架构，不新增框架或依赖。

## 验收标准

- 首页、创建流程、股票详情和便条弹窗视觉统一。
- 桌面和移动端无水平滚动、文字遮挡或不可点击控件。
- 原有 localStorage 数据可正常读取。
- 新建笔记、打开详情、主题切换和弹窗操作无 JavaScript 错误。
